vpngate
=======

vpngate.net .ovpn fetcher

### Usage

```bash
usage: vpngate [-arh]
-a : show all gates instead of best per country/protocol (last is best)
-r : refresh from vpngate.net

.ovpn files are stored in ~/.vpngate/vpngate_SCORE_IP_PROTO_COUNTRY.ovpn
```
