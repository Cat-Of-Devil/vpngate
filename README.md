vpngate
=======

vpngate.net .ovpn fetcher

### Usage

usage: vpngate [-arh]
-a : show all gates instead of best per country (last is best)
-r : refresh the list.txt file

.ovpn files are stored in ~/.vpngate/vpngate_SCORE_IP_COUNTRY.ovpn
